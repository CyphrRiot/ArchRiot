ArchRiot — Keybindings Help

Local quick reference. Open with SUPER+SHIFT+H. Website help remains on SUPER+H.

Basics

SUPER + H
Help (website)
SUPER + SHIFT + H
This keybindings help (local)
SUPER + D
App launcher (Fuzzel)
SUPER + RETURN
Terminal (Ghostty)
SUPER + SHIFT + RETURN
Floating terminal (overlay)
SUPER + L
Lock screen (Hyprlock)
SUPER + ESCAPE
Power menu
SUPER + SPACE
App launcher (Fuzzel)

Window Management

SUPER + W / SUPER + Q
Close window
SUPER + V
Toggle floating
SUPER + J
Toggle split
SUPER + P
Pseudo tiling
SUPER + TAB
Window switcher
SUPER + SHIFT + TAB
Fix off‑screen windows
SUPER + ←/→/↑/↓
Move focus
SUPER + SHIFT + ←/→/↑/↓
Swap windows
SUPER + CTRL + ←/→/↑/↓
Smart movement

Workspaces

SUPER + 1..4 (or 1..0)
Switch workspace
SUPER + SHIFT + 1..4
Move window to workspace
